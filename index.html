Portfolio

Rapport

Rapport is an open source application that automates messages and emails on facebook and gmail based on custom intervals. On the front end, I designed and styled the app using Bootstrap, HTML, and CSS. On the back end, I added a Giphy feature, set up the Kron job, linked the application with Google Gmail and Contacts API.

OkPluto

OkPluto, available on github is social networking app for dog owners and dog enthusiasts. The application features local search results for events, dog owners, and dog rentals. I created manage visitor and logged in states and created components to render them appropriately.

Persona

Persona is also an open source social networking application that matches users based on personality portraits derived from twitter feeds. I wrote the algorithm for determining users with similar personalities and integrated the IBM watson API.

ColorGen

ColorGen generates color schemes and lets designers see their designs in different colors. I created this entire app. The source code is available on gitHub.
